WDI Project 1

Salvy McMeme Face

The brief for this project was to build a game using JavaScript/jQuery, Css and HTML. For my game I opted to build an online version of The Gem Puzzle whereby the player is presented with a picture which is divided up into seven equally sized squares and one blank space. The tiles are shuffled at random and the user must rearrange the tiles by moving one tile at a time into the blank space until the picture is restored to its former glory. See the code and the README here. See the application here.

WDI Project 2

Vegan Restaurants in London

The imaginatively titled “Vegan Restaurants in London” is a RESTful Express application and utilises two external API’s. The application uses Google Maps to show the location of vegan restaurants in London and uses the Zomato restaurant API to obtain data on each of these eateries. Each of the restaurants found are marked on the map using a green ‘v’. The user is then able to click on this icon to view more information about this restaurant. See the code and the README here. WDI Project 3

Vibes

This is a MEAN stack application. As a group project we were required to use GitHub extensively to share/merge code. The application helps people exchange skills. Each user can post up something with which they require help and would then be able to ‘swipe through’ (in a Tinder-esque manner) other users’ profiles to see if there was any way in which they could help other users. The site includes a messaging service which users can use to communicate with one another when they had agreed to help each other out. See the code and the README here. See the application here

WDI Project 4

Lex Machine

This application uses an Angular front-end and a Rails back-end. It is an authenticated application designed to help the user expand their vocabulary. It invites the user to input a word for which they want more information. Once submitted the application populates that word’s page with etymological information from a dictionary API. It also allows the user to input and save information. For authentication I used JWT and Bcrypt and for the dictionary API I used Persons Dictionary API. See the code and the README here.
